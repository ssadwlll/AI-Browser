使用说明

1. 将 Token 填入 AI Browser 客户端的「设置」→「管理后台配置」→「后台 Token」
2. 或在 API 请求头中添加：Authorization: Bearer {token}
3. Token 有效期 7 天，过期后需重新登录获取
