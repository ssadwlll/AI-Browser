使用说明

1. AppKey 用于客户端连接服务端，替代旧的 Token 认证
2. 客户端在「设置」→「服务端连接」中填入 AppKey 和 AppSecret
3. 签名算法：HMAC-SHA256(appKey + timestamp, appSecret)
4. 请求头需携带：X-App-Key、X-Timestamp（秒级）、X-Sign
