## AI Browser - 产品功能说明文档

版本：2.0.0 | 更新日期：2026-07-02

---

## 产品概述

AI Browser 是一款 AI 驱动的浏览器助手 Chrome 扩展，以"智能对话 + 自主操作 + 脚本管理"为核心能力，帮助用户在日常浏览中高效完成信息提取、内容理解、表单填充和批量数据处理等工作。

**产品定位：** 让浏览器拥有"会思考、能操作"的 AI 助手，将繁琐的网页交互变成一句话就能完成的自动化任务。

**目标用户群体：**

- 信息工作者：需要频繁从网页中提取、整理、翻译内容
- 数据分析师：需要从网站批量采集数据并进行结构化处理
- 运营人员：需要定期监控网页变化、自动化重复性网页操作
- 普通用户：希望在浏览网页时获得 AI 辅助（翻译、解释、摘要等）

**核心价值主张：** 用户只需用自然语言描述需求，AI 助手即可自主规划任务、操作网页、调用脚本、汇总结果，全程可视化追踪进度。

## 核心功能详解

### 1. AI 智能对话

AI 智能对话是产品的基础功能，用户可以在侧边栏中与 AI 进行多轮对话，AI 能够理解当前网页内容并给出精准回答。

**使用场景：**
- 阅读外文文章时请求翻译
- 面对长文章时快速获取摘要
- 遇到不理解的概念时请 AI 解释
- 基于页面内容提出问题或请求分析

**操作方式：**
1. 点击浏览器工具栏图标或按快捷键打开 Popup，再进入侧边栏
2. 也可直接点击页面右下角的 AI 浮动按钮打开侧边栏
3. 在输入框中输入问题并发送
4. AI 流式返回回答，支持 Markdown 渲染和代码高亮

**支持的能力：**

- **多轮对话：** 保持上下文连续性，可追问、澄清、补充
- **页面内容理解：** 当用户提到"总结"、"翻译"、"提取"等关键词时，系统自动注入当前页面内容
- **文件上传：** 支持上传图片（需视觉模型）和 PDF 文件，AI 可直接分析文件内容
- **模型选择：** 从服务端提供的模型列表中切换，支持按供应商分组，标注视觉/工具能力
- **聊天记录管理：** 自动保存历史对话，支持新建对话清空上下文
- **快捷操作按钮：** 输入框旁提供"总结页面"和"智能工具箱"快捷入口

---

### 2. Agent 智能助手（自主网页操作）

Agent 是产品最核心的功能，一个能自主操作网页的 AI 智能体。用户只需用自然语言下达任务指令，Agent 会自动规划步骤、操作网页、调用脚本、汇总数据，并实时展示执行进度。

**使用场景举例：**
- "采集这个新闻网站的标题和链接列表"
- "提取页面上所有产品信息并整理成表格"
- "帮我在多个页面间导航并收集数据"
- "导出之前采集的结果"

**操作方式：**
1. 在侧边栏中点击导航栏的 "Agent" 按钮切换到 Agent 模式
2. 输入任务描述并发送
3. Agent 自动开始执行，实时展示步骤卡片和进度
4. 任务完成后 Agent 输出汇总结果，并附带完成度评估

**三阶段执行模型：**

Agent 采用独特的三阶段调度系统执行任务：

| 阶段 | 名称 | 能力 | 目标 |
|------|------|------|------|
| Stage 1 | 页面探索 | 15+ DOM 工具（点击、提取、导航、滚动等） | 获取页面结构信息和关键数据 |
| Stage 2 | 脚本处理 | 调用服务端远程脚本 | 批量处理数据（如逐页获取详情） |
| Stage 3 | 结果汇总 | 数据查询和输出 | 汇总所有数据，输出结构化结果 |

**任务规划与执行流程：**
1. Agent 自动读取当前页面内容和 URL
2. 自动搜索服务端工具库，匹配可用脚本
3. AI 创建三阶段待办列表（create_todo），系统自动校验合规性
4. 按待办顺序执行工具操作，系统自动追踪进度
5. 每个阶段完成后自动切换到下一阶段
6. 所有待办完成后调用 finish_task 输出汇总结果
7. AI 评判器对任务完成度进行事后评估

**进度追踪与可视化：**
- 侧边栏中实时显示步骤卡片，包含工具名称、参数、执行结果
- 页面内嵌的待办追踪面板（可拖动）展示三阶段进度条、完成百分比、当前待办状态
- 阶段切换时自动记录日志并在面板中展示

**安全与收敛机制：**
- 预算控制：70% 预算发出软警告，85% 触发紧急收敛
- 自动切换：Stage 1 连续 4 次无进展自动切换到 Stage 2；Stage 2 连续 3 次脚本失败自动切换到 Stage 3
- 工具幻觉拦截：验证工具名称是否在允许列表中，拦截不存在的工具调用
- 域名安全策略：导航操作受域名白名单限制
- 上下文压缩：消息超过 40 条时自动分层压缩历史消息

### 3. 划词工具

在任意网页上选中文本后，自动弹出操作工具栏，提供四种 AI 驱动的文本处理能力。

**使用场景：**
- 阅读外文内容时快速翻译
- 遇到专业术语时请 AI 解释
- 需要改写段落以改善表达
- 快速获取一段长文的要点摘要

**操作方式：**
1. 在任意网页上用鼠标选中一段文本（至少 3 个字符）
2. 选中文本上方自动弹出工具栏，显示四个操作按钮
3. 点击所需操作，系统自动打开侧边栏并展示 AI 处理结果

**支持的操作：**

| 操作 | 说明 |
|------|------|
| 解释 | AI 解释选中内容的含义 |
| 翻译 | 将选中内容翻译为中文 |
| 改写 | AI 改写选中内容，改善表达 |
| 摘要 | 提取选中内容的要点 |

---

### 4. 智能表单填充

AI 自动检测页面中的表单，分析字段类型和语义，生成合理的填充值并自动填入。

**使用场景：**
- 快速填写注册表单、调查问卷
- 生成测试数据填充表单字段
- 批量填写重复性表单

**操作方式：**
1. 当页面中存在 2 个以上可填写字段时，表单旁自动出现"智能填充"按钮
2. 点击按钮，AI 分析所有字段的标签、类型、占位符、选项等信息
3. AI 生成合理的填充值并自动填入各字段
4. 表单短暂高亮提示填充完成
5. 用户检查并修改后提交

---

### 5. 脚本管理

通过服务端管理和同步脚本，支持在网页上手动触发执行。

**使用场景：**
- 使用预置的高级数据处理脚本
- 在特定网站上执行定制化操作
- 管理和维护脚本库

**操作方式：**
1. 在 Popup 的"脚本"标签页中查看已同步的脚本列表
2. 点击"同步"按钮从服务端拉取最新脚本
3. 在侧边栏的"智能工具箱"中点击脚本执行
4. 执行结果通过回调机制反馈到侧边栏

**脚本同步机制：**
- 安装时自动同步一次
- 定时同步（可配置间隔）
- 手动同步按钮
- 脚本按域名匹配，按历史成功率排序

---

### 6. 定时任务

基于 Chrome Alarms API 的定时任务系统，支持周期性执行自动化操作。

**使用场景：**
- 每天定时打开指定网页检查更新
- 定期执行数据采集脚本
- 定时触发 Agent 任务

**支持的操作类型：**
- URL 导航：定时打开指定网页
- 脚本注入：定时执行指定脚本
- Agent 消息：定时向 Agent 发送任务指令

**操作方式：**
在侧边栏功能面板的"定时任务"标签页中创建、编辑和管理定时任务。

---

### 7. 网络请求捕获

在页面加载的最早阶段（document_start）注入拦截器，自动捕获页面所有的 XHR 和 Fetch 请求。

**使用场景：**
- 调试页面 API 接口
- 分析页面数据加载流程
- Agent 通过 capture_network 工具查询捕获的请求数据

**功能特点：**
- 捕获请求 URL、方法、状态码、响应预览
- 流式读取响应体（上限 16KB）
- 批量上报，减少性能开销
- Agent 可通过工具直接查询捕获数据

---

### 8. 任务模板

将常用的 Agent 任务保存为模板，下次可快速调用。

**使用场景：**
- 每日例行的数据采集任务
- 标准化的网页操作流程
- 团队间共享任务配置

---

### 9. 工具录制

录制 Agent 执行任务过程中的工具调用详情，形成会话记录。

**使用场景：**
- 调试 Agent 执行过程
- 复盘任务执行路径
- 为脚本开发提供参考

---

### 10. 数据导出

将 Agent 采集的数据或对话记录导出为多种格式。

**支持的格式：**

| 格式 | 扩展名 | 适用场景 |
|------|--------|---------|
| JSON | .json | 程序化处理、数据交换 |
| CSV | .csv | Excel 表格、数据分析 |
| Markdown | .md | 文档编写、Wiki 发布 |
| HTML | .html | 网页展示、报告分享 |
| 纯文本 | .txt | 简单文本、日志记录 |

## 界面说明

### Popup 弹窗界面

点击浏览器工具栏的扩展图标打开，提供三个标签页：

- **AI 助手：** 快速提问入口，提供"总结页面"、"翻译页面"、"提取数据"快捷按钮，所有操作引导至侧边栏
- **脚本管理：** 查看脚本列表、启用/禁用脚本、手动同步
- **设置：** 配置服务器地址、AppKey/AppSecret、模型选择、划词工具开关

### 侧边栏界面

产品的核心交互界面，采用 Chrome 原生 Side Panel：

- **聊天模式：** 与 AI 对话，支持 Markdown 渲染、代码块、文件上传
- **Agent 模式：** 自主任务执行，步骤卡片实时展示，工具调用结果可视化
- **导航栏：** 对话、Agent、脚本、设置等功能入口
- **功能面板：** 可拖动的浮动窗口，包含执行图谱、资源监控、任务模板、工具录制、定时任务、调试日志等标签页
- **数据导出：** 支持将对话记录和 Agent 结果导出为多种格式

### 页面内浮动按钮

在所有网页的右侧边缘显示一组浮动按钮（Shadow DOM 隔离，不影响页面样式）：

- **工具箱：** 打开侧边栏的智能工具箱
- **工具：** 工具快捷入口
- **待办：** 切换显示/隐藏待办追踪面板
- **设置：** 打开侧边栏设置
- **AI：** AI 功能入口

浮动按钮具有 MutationObserver 守护机制，即使被页面脚本意外移除也会自动重新注入。

### 任务进度面板

Agent 执行任务时，页面内嵌一个可拖动的待办追踪面板，实时展示：

- 当前阶段名称和进度百分比
- 三阶段待办列表及各待办状态（待处理、执行中、已完成、失败）
- 最近一次工具调用及结果
- 阶段切换日志

## 安全与隐私

### 请求签名机制
所有与服务端通信的请求均使用 HMAC-SHA256 签名，通过 AppKey + 时间戳生成签名头，确保请求的真实性和防篡改。

### 域名安全策略
Agent 的导航操作受域名白名单限制，阻止导航到未授权的域名，防止恶意跳转。

### XSS 防护
- 聊天内容使用 textContent 渲染，不直接插入 HTML
- 链接协议过滤（仅允许 http/https/mailto）
- HTML 转义处理所有用户输入

### URL 安全检查
阻止导航到危险协议（javascript:、data:、vbscript:、file:、blob:）和系统页面（chrome://、edge://、about:）。

### 回调验证
注入脚本的回调消息使用随机 Nonce 验证机制，防止页面脚本伪造回调消息。
